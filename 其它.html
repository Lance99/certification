各式效應總整理

  1. 霍桑效應：當被觀察者知道自己成為被觀察對象而改變行為傾向的反應
  2. 強(約翰)亨利效應：當控制組的受試者感受到他們處在和實驗組競賽情況下，有超乎尋常表現的現象
  3. 漣漪效應：教師訂正學生不良行為時，會影響鄰近學生的行為
  4. 月暈效應(以偏蓋全)：一個人表現好時，大家對他的評價遠遠高於他實際的表現，反之，一個人表現不好的時候，別人眼中所認為的差勁程度，也會遠大於他真正差勁的表現
  5. 刻板印象(以全蓋偏)：例如：男生都恐有武力
  6. 比馬龍效應(自驗預言)：教師對學生的期待與學生的表現成正相關
  7. 普力馬原則(=普莫克原則=阿媽法則=老祖母法則)(屬於條件契約)：使較不喜歡的活動與較喜歡的活動連結，來強化個體從事不喜歡活動的意願
  8. 閃光燈效應：指引人震撼的事件，容易使人留下深刻的記憶
  9. 舌尖效應：話說到嘴邊卻突然想不起來的現象，認知論解釋為形音意聯結困難所致
  10. 破窗效應：一個房子如果窗戶破了，沒有人去修補，隔不久，其它的窗戶也會莫名其妙的被人打破；一面牆，如果出現一些塗鴉沒有清洗掉，很快的，牆上就佈滿了亂七八糟
  11. 蝴蝶效應：一系統的某一端若發生了小幅變動，透過整個系統的加乘作用，很有可能會對另一端發生極大的影響
  12. 約拿情結：對成長的恐懼，害怕失敗，也害怕成功
  13. 彌賽亞情結：把自己的本質角色界定為人類的救贖者(個人神話)
  14. 哈姆柴克：維持學生學習動機的二大要件為良好的師生關係和培養和諧的氣氛
  15. 仰巴腳效應(出醜效應)：才能平庸者固然不會受人傾慕，而全然無缺點的人，也未必討人喜歡。最討人喜歡的人物是精明而帶有小缺點的人
  16. 旁觀者效應：危機現場中人數愈多，救助行為出現的可能反而愈少
  17. 萊斯托夫效應：學習材料中最為特殊的事件，容易記憶
  18. 時近效應：最近(最後)學習教材最容易記得
  19. 初始效應：最初學習教材最容易記得
  20. 過度辯證(overjustification)效應：外在的動機（如金錢上的獎勵）若是過於強大、明顯，此時我們內在的動機或是出發點受到外在的誘惑，會使學習者認知上失調，進而產生一種為的「只是錢」的理念
  21. 統計迴歸效應：受試者在前後測中，第二次測驗成績有向團體平均數迴歸的傾向稱迴歸效應。也就是說，再在第一次測試較差的學生可能在第二次測試時表現好些，而第一次表現好的學生則可能相反，這種情形稱為統計迴歸效應
  22. 寬心劑的效應：病人在不知情下服用寬心劑(藥劑代替品如維生素)，仍然發生治療的效果
  23. 柴嘉妮(Zeigranik)效應：未完成工作的會比巳完成工作容易記得
  24. 白金生定律：
    • 行政首長喜歡增加部屬，以形成自己的權勢
    • 機關開會時間長短，與議題重要性形成反比
    • 機關採用委員會愈多，效能愈差，而形成較小的核心小組以代替委員會功能
    • 行政效率日趨低落，但建築及辦公設備卻日益豪華機關可用的錢必想法用完
  25. 不完全控制定律：堂斯依據莫非定律提出推論，沒有人能夠完全控制大型組織的行為，因此需居安思危
  26. 麥爾士定律(職位決定立場)：本位擺中間，其他放兩旁；在什麼位置，說什麼話
  27. 不稀罕效應(BOHICA)：指組織成員對革新計劃的抵制態度，認為只要忍耐即可不受影響，久之革新計劃最後將無疾而終
  28. 破唱片法：像是唱片壞掉一樣,不斷播送同樣的話語(你在走廊奔跑， 有沒有違規？)
  29. 社會閑散效應：是指當個人的努力與別人的努力結合在一起的時候，他們工作起來就沒有分開來那麼起勁
  30. 原初效應：呈現正反兩面資料且企圖使正面資料印象加深時，應先呈現正面資料，使學生對正面資料產生先入為主的觀念

常考申論題

教師作為轉化型知識份子」的兩項重要意義，再列舉我國學校教師可有的三種作為

  1. 教師須帶領學生養成批判的能力,透過反省,以利於未來學生出社會後,所遇到不公不義之事件,付諸轉化的行動
  2. 強調教師的主體性,不只是傳統社會中傳遞知識的工具,而是可以透過知識的轉化,達成利社會的行動
  3. 提供學生反省批判的機會,透過教學課程設計,讓學生多去內省和反思
  4. 強調課程中是經驗的重組,聆聽學生的聲音和意見。
  5. 肯定學生的特質,幫助他們自我認同,並培養其思考能力,進而去付諸行動,以消除社會上的不平等

教師宜採取哪些教學策略以照顧到學生的個別差異？

  1. 編序教學法：使用習題簿、教科書或電腦系統教導學生，讓學生可以馬上核對答案，獲得立即性的回饋，適應個體差異，並以學習者的經驗不斷優化教材。
  2. 凱勒計劃（PSI):學生學習自我控速，依照自己的性向、速度和時間，老師利用形成性評量和總結性評量得知每個個別差異，以進行補救教學。
  3. 精熟學習法：以學生是否通過考試來判斷是否達成熟練標準之要求，若未達成標準，則再學習，再考試，直到通過才進行下一關單元。
  4. 道爾敦計畫：即根據柏爾的自我練習、自我測驗和校正的個別學習原理加以修正而來。因此，在學校組織上也都打破年級的限制，以適應學生的不同學習速度。
  5. 拼圖法
  6. 多元評量
  7. 補救教學

提升自我概念方法(適用於被關係霸凌者、有自殺傾向者)

  • 意義治療法
  • 完形治療法
  • 柏恩斯(burns)提出協助當事人發展自我概念的七種方法
    • 提供成功的機會，並確信對當事人的輔導與要求能切合個人的潛能
    • 顯現對當事人的興趣與對當事人無條件的接納，如微笑、問候、談話
    • 注重個人積極的層面，不要強調失敗與缺點
    • 多鼓勵，少批評與嘲諷
    • 對特定行為加以指正，但勿批評整個人
    • 預防他們因恐懼失敗而害怕作新的嘗試
    • 喜歡當事人做有價值的常識並給予信心
  • 給予正向期待、回饋(自我應驗預言、畢馬龍效應)
  • 找出優勢、建立目標
  • 分享成功經驗
  • 營造鼓勵不批評的環境

培養挫折容忍力的方法

  1. 協助其抒發情緒的管道
  2. 學習請他人幫忙及協助
  3. 給予對過程努力的肯定勝於結果的完美
  4. 平時多給予正反經驗，培養自信及體驗不如意的狀態

教師提高學生自我效能作法

  1. 恰當進行外部強化
  2. 重視學習策略的培養
  3. 幫助學生經常體驗成功
  4. 樹立榜樣：把學生的註意力吸引到榜樣身上，使學生從榜樣身上看到自己的潛能，增強自我效能感
  5. 創造寬鬆和諧的教育環境
  6. 引導學生自覺言行自我強化：首先必須建立自我行為標準，然後根據設立的標準來評價、檢查自己的行為及效果
  7. 幫助學生進行正確的歸因訓練

課程評鑑需考慮的事項

  1. 課程評鑑的目的：診斷課程方案的缺點或問題、修訂與比較課程、選擇課程方案、及改善課程設計及實施的績效。
  2. 課程評鑑的範圍：課程教材、教學計畫、實施成果等。
  3. 課程評鑑的評鑑人員：結合內外部人員，因為內部人員對課程方案較為瞭解而外部人原有專業的評鑑訓練且公平客 觀。
  4. 課程評鑑的實施方式：會議討論、書面評估、面談對話或實境觀察。
  5. 評鑑結果的利用：應做有效的利用，包括改進課程、編選教學計畫、提升學習成效，以及進行評鑑後的檢討。

面對情緒管理問題學生輔導方式

  • 對自己

    • 適當表達自己情感
    • 且慢發作：當情緒來臨時，應將感覺速度放慢。可在心中默數十秒，並深呼吸，來減緩速度
    • 紓解壓力
    • 面對逆境：找出自己為何情緒不穩的原因，面對它並有效地解決
  • 對他人
    • 同理心
    • 包容力

認知方面：駁斥非理性信念、引導理性信念
情緒方面：幽默運用、角色扮演、回應感受

教師在教學上欲使學生達成良好的學習成效, 其應考量要素如下

  1. 準備原則：教學起初要能引起學生的好奇心和學習動機, 使學生對將學習的課程做好身心上的準備
  2. 適性原則：教材和教法要能配合學生的能力與經驗來做適當的選擇與安排
  3. 環境要素：若要使用多媒體設備或實驗器具, 要考量教室環境是否具備與事宜.
  4. 評量回饋：針對教學是否達成成效. 需要有評量的回饋, 所選用的教法應能發展適當的評量機制
  5. 專業知能：身為教師欲達成良好的教學成效, 在專業知識, 教材運用,編排和呈現, 都要有專業的訓練和知識背景

學生背景漸趨多元教師應具備哪些多元文化素養

  • 增進教師素養：教師應接觸多面向的新知事物，培養自我多方學習的能力
  • 具備同理心
  • 培養國際觀點
  • 不同觀點的課程設計
  • 養成社會實踐力：身為一位老師可以試著從小處去改變，從自身做起，再為小的事情也可以累積成巨大改變

對於行為異常學生

  • 詢問學生的感受，了解他目前的狀況
  • 進行班級宣導並請求班上同學協助
  • 視情況嚴重程度轉介輔導室，尋求專業的協助

霸凌事件處理

  • 受霸凌者

      • 無條件積極關懷
      • 同理心
    • 培養解決問題的能力
    • 進行社交技巧訓練
    • 給予成功經驗提升自我效能
    • 深入了解霸凌者組成、霸凌產生的原因，據此擬定因應策略
  • 霸凌者
    • 現實治療法-角色扮演：讓霸凌者扮演被霸凌者的角色，使其能夠感受到被霸凌者的心情，進而發現自己行為的不當
    • Kohlberg的道德兩難討論：教師可以安排適當的議題討論，讓學生練習思考並討論，並運用加一法則，協助學生提升至高一層次的道德觀。
  • 輔導班上其他同學
    • 現實治療法-角色扮演，加入一個旁觀者的角色，讓班上其他同學感受被旁觀的、被忽視的痛苦，進而明白不是旁觀就好，應該要伸手幫助別人
    • 營造正向接納的班級氣氛
    • 大家共同思考可行的因應策略及理想作法

美國教育學家杜威(J.Dewey)的知識論之要點及其教育主張

  • 杜威的知識論要點

    1. 知識產生於經驗，經驗是生物與環境交互作用的結果。
    2. 知識具有三種特質：實用性、行動性及創造性。

  • 杜威的教育主張

    1. 教育的本質：教育即生長，就是經驗不斷的累積及重組改造。教育是動態的，一種生長的歷程。
    2. 教育的目的：教育會讓人成長，教育本身學習的歷程就是教育的目的。
    3. 教育的方法：
      • 從做中學
      • 問題教學法：提供情境讓學生解決。
    4. 教育的內容：
      • 把實際生活的題材納入課程中。
      • 連續性的教育概念
        • 連續性是指前後兩個經驗的相關性
        • 依賴經驗的交流，個人經驗才可持續的重組與改造。
      • 問題解決的教育概念
        • 知識的實用性：能夠解決當前困難的知是最實用
        • 學校及社會，社會即生活
        • 「做中學」的先決條件，學生要自動自發的學習，因此教師必須了解學生的心理，善用各種方法引導學生學習。
