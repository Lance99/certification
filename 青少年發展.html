青少年發展

青少年發展的生理基礎

神經系統

  • 前腦
    • 大腦皮質
    • 視丘：傳遞脊隨的神經衝至大腦皮質
    • 下視丘：自主神經系統控制中心 例如：內分泌
    • 腦垂腺：內分泌腺
  • 中腦：視覺與聽覺的反射中樞
  • 後腦
    • 延腦：控制呼吸、心跳、消化等
    • 腦橋：睡眠品質
    • 小腦：身體自由活動能力

大腦半球構造與功能

  • 大腦皮質構造
    • 額葉：學習、記憶、思維、協調各部位活動
    • 顳葉：聽覺、語言
    • 頂葉：軀體感覺
    • 枕葉：視覺
  • 大腦分區構造
    • 語言區
      • 不羅卡區：發音
      • 為尼克區：理解
    • 大腦左半球主宰個體的意識行為，右半球的意識反應來自左半球的輔助
    • 大腦側化造成左撇子
    • 海馬迴：記憶中樞
  • 內分泌系統或腺體的構造與功能
    • 腦垂腺：分泌生長激素、腺性激素、泌乳激素等
    • 甲狀腺：可促進全身細胞氧化作用，維持身體生長
    • 副甲狀腺：血液中鈣與磷的濃度，以維持神經系統與肌肉的興奮劑
    • 松果腺
      • 可調整個人生理週期
      • 個體幼年時期，松果腺體積最大，青春期後逐漸會退化
    • 胸腺：可產生T細胞
    • 下視丘：控制個體的動機、情緒、愉悅與痛苦

感覺與知覺發展

  • 感覺(生理變化)：個體根據身體感官接受與辨認環境刺激訊息之歷程
  • 知覺(心理變化)：個體根據身體感官接受與辨認環境刺激訊息，由腦針對訊息加以解釋的歷程

    感覺是知覺形成的基礎

青少年的認知發展

Piaget認知發展理論

  • 強調成熟，不強調學習
  • 適應(包含同化與調適)：個體因環境影響而改變心理歷程
    • 同化：不改變原基模，僅將所接收的訊息納入其認知結構中
    • 調適：個體原基模不能直接同化新知識時，調整其基模以新收新知識的歷程

✍ Piaget認為「適應」也是一種智力的展現

期別 年齡 認知特徵
感覺運動期 0~2歲 對物體認知具有物體永存的概念
前運思期 2~7歲 能使用語言表達(但具自我中心傾向)
能使用符號代表實物
能思維但不合邏輯
具體運思期 7~11歲 能根據經驗解決問題
理解可逆性
理解守恆
去中心化
已具備分類能力(但無類包含思維)
形式運思期 11歲以上 具抽象思維
能案假設驗證解決問題
能案形式邏輯思維解決問題
  • 前運思期兒童思維之所以不合邏輯的三因素：中心化(無守恆概念)、不可逆性、自我中心化(無法客觀看世界e.g., 三山實驗)
  • 類包含：指能分辨主類與次類 e.g., 紅花包含於花

  • 對Piaget認知發展理論的評述

    • Piaget認知發展論缺少社會文化意義

    • Piaget的認知發展理論不主張藉由學習加速兒童認知發展

    • Piaget低估了兒童的思維能力、高估了青少年的思維能力

    • Piaget的四階段理論無法涵蓋整個認知發展的全部歷程
    • 成人的思維往往不只形式邏輯思維去尋求唯一解，而是採辯證思維，權衡得失(因此加上第五階段：後形式思維) ☛ 新皮亞杰主義

維果茨基(Vygotsky)的認知發展理論

Vygotsky的認知發展基本概念

基本概念 相關說明
認知發展 人從基本心理功能轉化為高級心理功能之歷程
社會環境是影響認知功能的重要因素
語言是促進個體認知發展的工具
心理功能 基本心理功能：人類與動物共有的
高級心理功能：人類獨有的(語言、邏輯推理、思維等)
概念 科學概念與自發概念
發展歷程：隨意期、雜思期、潛在概念、真正概念
自我中心語言 可促進個體認知發展的功能
引導式參與 例如：跟著學長學街舞

  • 改善人類生長環境，可以促進兒童智力發展

  • Vygotsky不同意Piaget前運思時期的自我中心語言觀點(學前兒童自我中心語言只代表兒童對事物認知思維的一種表達方式，與其認知發展無必然關係)

    • Vygotsky：語言是習得的心理工具
    • Chomsky：人類語言具有深層的結構
    • Piaget：思考有助於語言與符號的發展
  • 近側發展區與鷹架作用
    • 近側發展區(zone of proximal development)：實際認知發展水準到他可能認知發展水準間的差距(近側發展區上限代表兒童認知發展的最大潛力)
    • 可能認知發展水準：是指旁人協助下可以達到的成就
    • 鷹架作用：在個人ZPD內旁人給予的協助稱之

青少年的社會發展

家庭對青少年的發展影響

  • 依附關係[代表：Bowlby]
    • 焦慮型依附：具低自我認同感，雖想尋求同儕接納缺難以和同儕建立親密關係
    • 安全型依附：與父母有親密依附關係，可以獲得協助
    • 逃避型依附：通常自尊較低
  • 衛星理論[代表：Asusbel]
    • 青少年逐漸發展成熟時，自主性增加，開始產生「脫離衛星化」與「重新衛星化」現象
    • 青少年價值迷失時，產生「非衛星化」現象，無法建立衛星化親子關係
  • 家庭星座理論[代表：Adler]
    • 父母象徵太陽，子女象徵星座
    • 不同出生序青少年會有不同人格特徵

社會認知與互動

相關理論

  • 歸因論[代表：Weiner]
    • 內在歸因(=性格歸因)
    • 外在歸因(=情境歸因)

✍ 習得無助感(≠失敗的學生，只是歸因)[代表：Seligman]：將成功歸因於努力，將失敗歸因為能力不夠

  • 共變論[代表：Kelly]
    • 情境與性格間共變的關係
    • 判別社會行為的歸準
      • 社會行為的共同性
      • 社會行為的一致性
      • 社會行為的情境特殊性
  • 社會認知論(角色取替理論)[代表：Selman]
    • 階段0：短暫互動
    • 階段1：單向協助
    • 階段2：順境中合作
    • 階段3：親密與分享(一般少年對友誼的看法)
    • 階段4：自主又相互依賴(維持良好友誼需付出也需信守承諾)
  • 基本歸因誤差：個體對於他人行為的解釋往往偏重內在歸因忽略外在歸因
  • 自利歸因偏差：個體對於自己失敗行為的解釋往往偏重外在歸因，對於自己成功的行為往往偏重內在歸因

態度

相關理論

  • 說服性溝通
  • 認知失調理論[代表：Festinger]
    • 當對於某件事的相關認知失衡時，個人就會產生改變認知的驅力，以改變自己的態度
  • 學習理論
    • 個人態度形成及改變，可經由學習來獲得
    • 學習理論觀點：行為、認知、觀察、人本學派
  • 態度平衡理論[代表：Heider]
    • 個人態度是否平衡取決於個人、他人與態度間的交互作用
  • 自我知覺理論[代表：Bem]
    • 當個人不明了本身態度情況時，會以旁觀者的角度來解釋自己行為及推論本身態度

艾瑞克遜(Erikson)心理社會期發展論

  • 心理社會期八階段論
年齡 期別 正向發展 負向發展
0~1 嬰兒 信任 不信任
1~3 幼兒 獨立 羞怯懷疑
3~6 學齡前兒童 主動 內疚退縮
6~11 學齡兒童 勤奮 自卑
11~18 青少年(青春期) 自我統整 角色混亂
19~30 成年早期 友愛親密 疏離
31~50 成年中期 積極生產 自我頹廢
50up 成年晚期(老年) 完美無憾 悲觀絕望
  • 與Freud都屬精神分析學派
  • 與Freud理論相異處：
    • Erikson不是基於人格異常者的心理特徵所建構
    • Erikson不同於Freud持早期決定論，而是將社會全程視為連續不斷的人格發展歷程
    • Erikson的理論以自我的內在成長因素作為社會發展動力，是社會性的，所以它的理論所解釋的人格發展可視為心理社會發展，而Freud的只可視為個體的性心理發展

✍ Erikson理論後續的研究

  • Marcia按自我統合的程度將大學生分四類，表示大學生在統合過程中，自我發展達到的心理狀態(統合狀態)
    • 迷失型：尚未認真思考過人生方向
    • 未定型：有思考過但尚未確認人生方向
    • 定向型：有認真思考也確認人生方向
    • 早閉型：未經過統合危機，對未來也沒有認真思考過，多半將自己的人生交由長輩處理，不關心現在也不在意未來
  • 個體的自我發展需受到社會約束不能順其自然的任其自由發展
  • 全人教育：知、行、情、意四層面的心理歷程

✍柯爾伯格(Kohlberg)(道德發展階段論)

  • Kohlberg與Piaget理論有頗多相似處，都採用認知發展取向，但是K的理論更能引起心理學領域的重視
    • Kohlberg排除了傳統上道德中的分類觀念(有/無或是歸類)，而是每個人的道德都是隨年齡經驗的增長而發展
    • 道德判斷不純是個是非問題，而是在面對道德問題情境時，個人從各方面綜合考量所做的價值判斷
  • Kohlberg與Piaget最大相似處是他們都採取心理發展的觀點解釋人類道德心理的形成
    • 十歲前：自律道德(有中心化取向)
    • 十歲後：他律道德

青少年生涯發展

青少年生涯發展相關理論：

生涯發展論[代表學者：舒波(Super)]

  • 青少年生涯發展受到種種因素影響：家庭、學校、同儕、社會道德...
  • 青少年階段最重要的發展重點工作：多了解自己的興趣和能力以及世界工作的面貌
  • Super的生涯發展階段(生命彩虹圖)：
    1. 成長期(0-14)
    2. 探索期(15-24)
    3. 建立期(25-44)
    4. 維持期(45-65)
    5. 衰退期(65up

發展理論[代表學者：Havighurst]

  • 青少年階段是生涯實驗期，開始對工作產生興趣，並可以統合個人能力與價值觀
  • 主張個人生涯發展分為四階段：
    1. 追求重要他人認同
    2. 養成工作習慣
    3. 認同真實工作
    4. 成為具有社會生產力的個體

動機理論[代表學者：Maslow]

  • 當個體較低階段的需求度獲得滿足後才會依序出現較高階段需求的滿足

社會學習取向的生涯決定論[代表學者：Krumboltz]

  • 影響個人生涯決定因素：遺傳、環境、學習經驗、情緒、解決問題能力
  • 以偏概全的類推：即以自身想法、興趣或他人期待所形成的理想當做目標來追求

青少年團體輔導及學習

  • 團體工作發展階段

    1. 初期階段：定向與探索、尋找意義、建立信任、團體形成
    2. 轉換階段：凝聚成員對團體向心力(處理抗拒、衝突、支配)
    3. 工作階段:凝聚力與效能
    4. 結束階段：讓成員能夠在現實生活中實踐(鞏固與結束)

  • 催化：用以增加團員們在團體中有意義的互動 e.g., 協助釐清目標...

  • 教師在班級中進行學習發法的輔導策略：
    1. 分享自身學習經驗方法
    2. 協助學生了解自己的學習方法、訂定改善計畫、提供回饋
    3. 直接教學生有效的學習方法並實作
    4. 面談學生(最佳面談座位安排為90度座位)
    5. 鼓勵學生的獨特想法
  • 輔導方案的系統計畫模式執行步驟
    1. 組織輔導小組
    2. 蒐集相關資料
    3. 做成輔導決定
    4. 建構輔導計畫
  • 對被忽略型及被拒絕型學生的輔導策略
    1. 教師可安排小組討論或團體遊戲等教學或動，讓被拒絕型及被忽略型學生有機會與其他學生互動，增加彼此了解
    2. 安排他們做班級幹部或鼓勵其在課堂中發言，並給予正向回饋以讓其願意放下心防融入班級團體中
    3. 請班上受歡迎人物帶領他們參與班級事物，提升他們的同儕接受度

    1. 立即向當地主管機關通報(不得超過24hr)，而關於被害人個資應與保密

✍金滋伯(E. Ginzberg)生涯發展理論

  1. 幻想期(0-11)：幻想遊戲是未來工作的依據。
  2. 探索期(11-18)：又可分4個時期
    1. 興趣發展階段：11~12 ，開始覺察並培養某些職業的興趣
    2. 能力培養階段：13~14 ，以個人為核心，培養自已興趣方面的能力
    3. 價值階段：15~16，開始了解職業價值，並考慮社會及個人的需要
    4. 轉換階段：17~18 ，統整所有有關職業選擇之資料，正確了解其未來的方向
  3. 現實期(18up)：興趣和能力的整合，價值觀進一步發展，促使"職業選擇特定化"

✍發展螺旋論(葛賽爾（Gesell)

  • 生長是螺旋具有前進後退的律動現象，形成了螺旋狀逐步爬升，因此稱之為「發展螺旋論」。
  • 認為個體的發展是有次序的，遺傳所導引的成熟才是成長的基本機制，環境的力量影響微乎其微。
  • 生物力量是青少年發展的主導因子，因此優生保健的重要性甚於後天環境的教養。
  • 青少年不一定會產生暴亂與麻煩，時間自然會解決個體發展過程中的困擾。
