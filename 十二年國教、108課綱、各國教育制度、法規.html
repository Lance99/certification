美國教育制度

  • 美國重要教育政策
    • 「不讓任何孩童落後」
    • 「閱讀方案」
    • 「教師措施」
  • 特許學校：以高風險或少數族群學生為教育對象
  • 中等教育不實施分流(技職非技職)
  • 「提早教育」制度是針對：遭文化剝奪的兒童

日本教育制度

  • 高等教育分三級結構
    • 第一級：短期大學、高等專門學校、專修學校
    • 第二級：綜合大學、多科大學與單科大學
    • 第三級：研究所
  • 教育行政特色
    • 中央集權教育行政運作，但在教育財政上卻是地方分權
    • 與美國相同，成立教育委員會
    • 重視終身教育、尊重個性

英國教育制度

  • 教育行政特色
    • 政府對教育的干預增加，中央集權色彩漸濃
    • 學校在經費與人士權力增加
    • 以就業為教育改革導向，重視績效管理
  • 英國中等教育機構(雙軌制：貴族、平民) → 台灣是單軌制
    • 文法中學：招收智力與學力較高的學生，為大學的學術研究奠立基礎
    • 技術中學：提供科學及工藝的應用課程
    • 現代中學：非屬於文法或技術中學的學生則進入此學校
    • 綜合中學：勞工階級、就業取向(綜合文法、技術及現代中學為一體)
    • 公學：屬於傳統私立貴族學校
    • 預備學校：7-13歲
  • 「教育行動區」：改善學校的不利因素，提升學生學習表現
  • 「第六級學院」：為升大學做準備
  • 夏山學校：以兒童為中心經驗主義取向的學校
  • 高中畢業後須考「會考」，根據此成績申請大學

法國教育制度

  • 學校制度
    • 10年義務教育
    • 中等教育中有學徒制度
    • 高中畢業後須考「會考」，根據此成績申請大學
  • 教育行政特色
    • 教育事務方面重視民意參與
    • 教育行政組織專業分工，各司其職
  • 法國中等教育有學徒制系統
  • 法國的教育行政層級分3級：中央、大學區、省(府、郡(縣市))
  • 每個大學區設總長一人，大學區總長由教育部長提名，總統任命

德國教育制度

  • 學校制度
    • 9年或10年義務教育(看地方)
    • 中等學校
      1. 2年「定向階段」(定向學級)【為避免過早分化(技職非技職)】
      2. 中等教育第一階段：主幹學校、實科中學、文科中學、綜合中學
      3. 中等教育第二階段：文科中學高級部、職業學校
    • 中學階段實施「職業義務教育」
    • 德國學制中，在分化後2年有定向階段
    • 高中畢業後須考「會考」，根據此成績申請大學

芬蘭教育制度

  • 教育理念
    • 教好每一位孩子
    • 強調終身學習能力培養
  • 教育制度特色
    • 中小學階段免學費
    • 設有課後輔導機制
    • 教學中以學生為主體、採主題式統整教學

十二年國教

兩大願景

  • 適性揚才
  • 終身學習

三大課程發展理念

  • 自發
  • 互動
  • 共好

四大課程目標

  • 啟發生命潛能：提升學習動機(自發)，增進自我價值感
  • 陶養生活知能：培養基本知識，加強人際溝通與互動(互動)力
  • 促進生涯發展：讓學生學會如何學習，培養終身學習的意願與能力
  • 涵育公民責任：厚植民主素養，致力於生態永續的共好(共好)理想

五大理念 (口訣：有因適多優)

  • 有教無類
  • 因材施教
  • 適性揚才
  • 多元進路
  • 優質銜接

以核心素養做為課程發展主軸

【核心素養】：指一個人為適應現代生活及面對未來挑戰所應具備的知識、能力、態度，學習不應以科學知識或技能為限，應與生活結合，透過實踐力彰顯學習者的全人發展

基本理念(三面九項)

核心素養面向及核心素養項目

  • 自發
    • 身心素質與自我精進
    • 系統思考與解決問題
    • 規劃執行與創新應變
  • 互動
    • 符號運用與溝通表達
    • 科技資訊與媒體素養
    • 藝術涵養與美感素養
  • 共好
    • 道德實踐與公民意識
    • 人際關係與團隊合作
    • 多元文化與國際理解

教育政策的決定六階段：

  1. 認識問題界定目標：確定明確的計畫目標
  2. 設定判斷的標準：考慮法規、價值、人力等面向判斷標準
  3. 蒐集相關的資料：以客觀詳細的資料為依據，以便解決問題
  4. 擬定各種可行的計畫：為挑選最具經濟效益的計畫，可使用腦力激盪法擴散性思考
  5. 選擇最佳方案：以目標有效程度、可行性、經濟程度作為選擇的三標準
  6. 實施及評鑑改進：過程中以形成性評量；結束以總結性評量檢核政策執行是否有改進空間

國民中小學補救教學

接受補救教學的學生以不超過全校各科目總受輔人數之35%為原則

各式法規

教育基本法(教育的憲法，不論學生教師家長的基本權力義務都在此規範)

  • 保障人民學習及受教育之權力
  • 政府鼓勵私人興學
  • 教育人員之工作、待遇、進修學歷義務的保障【??可能是教師法】
  • 家長有輔導子女的責任
  • 人民享有請求學歷鑑定之權利
  • 保障學生學習權、受教權、身體自主權、人格發展權
  • 教育經費、教育事務審議、諮詢、協調
  • 禁體罰
  • 實驗教育三法的法源基礎

教師法(教師的工作權力義務)

  • 明定教師的權力義務
  • 保障教師工作與生活
  • 教師資格檢定、審定、聘任、待遇、進修、撫卹、退休等
  • 高級中等以下學校教師初聘為一年；續聘第一次為一年；以後續聘每次為二年；續聘三次以上成績優異者，經教師評審委員會委員2/3以上審查通過後，得以長期聘任，聘任期由各校評審委員會統一訂定

國民教育法(行政規則有關的事項)

  • 六至十五歲孩童應受國民教育
  • 資優跳級以一年為限(國小才能跳級)
  • 鼓勵私人興辦學校
  • 訂定國小課程
  • 訂定學、班級規模組織
  • 常態編班、分組學習

兒童及少年福利法

12-18歲

12歲以下算是兒童

少年事件處理法

至多到21歲

校園霸凌防制準則

疑似校園霸凌事件之被霸凌人或其法定代理人，得向行為人(施害者)所屬之學校申請調查；學校於受理申請後，應於三日內召開防制校園霸凌因應小組會議，開始處理程序，並於受理申請之次日起二個月內處理完畢

高級中等教育法

  1. 將高級中等學校概分為普通型、技術型、綜合型、單科型四種類型

    單科型:採取特定學科領域為核心課程，提供學習性向明顯之學生，繼續發展潛能之學校 EX 花蓮縣體育實驗高中(花蓮體中)

  2. 103 學年度各就學區免試入學之總名額，應占核定招生總名額 75%以上，並逐年提升，至 108 學 年度，應占核定招生總名額 85%以上【即：以免試入學為主】

    免試入學仍然可能會採計會考成績，只是入學前不必再考其他測驗

  3. 規定中央主管機關得參與高級中等學校課程綱要及其實施之有關規定的訂定
  4. 主管機關:在中央為教育部;在直轄市為直轄市政府;在縣(市)為縣(市)政府

其它

  • 資遣：沒有適任工作所以資遣，發給資遣費，但是不等於有違法或犯錯！

  • 停聘：「暫時」停止聘約，可能是因為有涉嫌違法，正在調查中。若調查後無罪或經申訴成功，則學校仍需發給當事人停聘期間(沒有工作)的薪水
  • 轉型領導：係指領導者會以前瞻性的遠景以及個人的魅力，運用各種激勵策略，
