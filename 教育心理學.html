心理學的學派流別

一、科學時期五大學派

I. 結構學派

代表：Wundt, Titchener

  • 研究意識結構
  • 反對哲學思辨的心理學研究
  • 心理經驗經過內省後 → 意象、感覺、感情

II. 功能學派

代表：James, Dewey

  • 了解人如何憑意識經驗對其生活環境做出有效適應
  • 心理學研究應著重心智發展的目的與功能

III. 行為學派

代表：Watson

IV. 完形學派

代表：Wertheimer, Kohler

  • 以意識經驗為研究主題，但研究目的的旨在探討意識的整體性與組織功能
  • 感覺透過【心】 → 統合的意識經驗
  • 屬於整體論，反對結構學派與行為學派的元素論

V. 精神分析學派

代表：Freud

  • 人格發展的五階段 (同Erikson心理社會期八階段論前五階段)
年齡 期別 正向發展 負向發展
0~1 口腔期 信任 不信任
1~3 肛門期 獨立 羞怯懷疑
3~6 性器期 主動 內疚退縮
6~11 潛伏期 勤奮 自卑
11~18 兩性期 自我統整 角色混亂

二、現代科學心理學五大取向

I. 行為論取向

  • 古典制約(被動的) = 反應性反應
  • 操作制約(主動的) = 操作性反應

II. 精神分析論取向

  • Freud
  • neo-Frued(加入社會文化因素)

III. 人本論取向[代表：Maslow, Rogers]

  • 自由意志與決定論之爭議
  • 人性本善，以正常人生活中的需求抑制、價值等為研究主題

IV. 認知論取向

  • 探討知識來源問題、融合行為學派與完形學派觀點
  • 以【感覺】、【注意】、【知覺】、【意識】、【學習】、【記憶】等內在心理歷程為研究主題

V. 心理生物學取向

行為主義取向的教育心理學

行為主義的學者

  • Watson
    • 利用狗唾液實驗(由Pavlov所做的實驗)解釋人類學習行為
    • 古典制約包括：非制約刺激、非制約反應、制約刺激、制約反應
  • Thorndike
    • 教育心理學之父
    • 不屬於行為主義學派而屬於聯結主義
    • 提出「學習三律」
      • 練習律：練習次數越多則個體反應與刺激間的連結越強
      • 準備律：個體越有動機則越可以展現預期的學習行為
      • 效果律：個體反應後，若可得到滿足，其反應行為將會增強
    • 學習是個體在刺激情境中所產生的【刺激-反應】連結
    • 學習遷移【同元素論、形式訓練說】(屬於：聯合論=行為聯合論)
      • 正遷移
        • 水平遷移：擴大範圍、舉一反三、觸類旁通
        • 垂直遷移：突破既有的知識層面而提升到更高的思想境界
        • 高低徑遷移：
          • 低徑遷移：反覆練習而產生自動化遷程，屬動作練習、低層次認知能力。
          • 高徑遷移：原理原則的遷移，屬高層次的認知能力
        • 前導回溯遷移：
          • 前導式遷移：將目前所學有意用到未來的情境
          • 回溯式遷程：面臨問題情境，回想過去曾學習原理原則
      • 負遷移(干擾)
        • 順攝抑制：舊的影響新的
        • 倒攝抑制：新的影響舊的
  • 同元素論(桑代克)學加法有助於乘法、學中文有助於日文學習
  • 共原則論(賈德)水下射擊,甲組教水中折射原理,乙組未教,甲組進步較快
  • 轉換論=關係論(完形心理學)球場如戰場,在球場團體合作的環境與精神如同戰場般 </blockquote>
  • Skinner
    • 操作制約論：利用愉快與不愉快來改變行為(e.g., 一個人在行為後若出現愉快的結果，此人表現該行為次數就會增加)【正增強、削弱、塑造...等可供教學時採用】
    • 主張環境決定論，且認為後效強化是學習的重要關鍵
    • 立即強化優於延宕強化；部分強化優於連續強化
  • Bandura
    • 社會學習論(=三元學習論)：環境、個人、行為
    • 觀察學習：分為四階段(注意、保留、再生、動機)
    • 替代學習：透過他人成功或失敗來學習
    • 楷模學習(=模仿)
      • 直接模仿
      • 綜合模仿：學習者經模仿歷程而學得的行為，未必直接得自楷模一人，而是綜合多次所見而形成自己的行為
      • 象徵模仿：學習者對楷模人物所模仿者，不是具體行為，而是他的性格或行為所代表的意義。電影、電視、兒童故事中所描述的偶像型人物
      • 抽象模仿：習者觀察學習所學到的是抽象的原則，而非具體行為。算術解題時，學生從教師對例題的講解中，學到解題原則

        經由賞罰學習歷程幫助學生建立道德行為也是屬於社會學習論範疇

  • Meichenbaum
    • 自我增強
    • 自我調整學習模式
    • 認知行為改變技術：以行為和認知的原理為基礎，透過自我談話和自我教導來改變自己

行為主義取向學習論的相關概念

  • 普力馬原則(=普莫克原則=阿媽法則=老祖母法則)
  • 正增強：即特定行為後，滿足此人的需求
  • 負增強：即特定行為後，消除此人不想要的結果　例如：將功贖罪
  • 懲罰(呈現型懲罰)：即特定行為後，給予此人不想要的結果
  • 撤銷(撤離型懲罰)：即特定行為後，取消該人想要的需求

認知主義取向的教育心理學

知識學習(認知心理學觀點)

  • 陳述性知識
  • 程序性知識(知道如何做某事的知識)
  • 後設認知知識(對知識認知和覺察的能力)

認知結構論

意義學習論[Asusbel]

  • 學習應配合學生的認知結構
  • 教學步驟
    1. 提供前導組織
    2. 呈現學習材料
    3. 以【漸進分化】及【統整調合】原則完成教學

發現式學習論[Bruner]

  • 認知表徵發展階段
    1. 動作表徵(0~2歲)：用動作認識世界
    2. 形象表徵(3~5歲)：兒童透過心象來認識世界
    3. 符號表徵(6~7歲)：運用抽象符號來認識世界
  • 強調學生主動發現知識的可能性，重視學習情境結構
  • 教師應配合學生經驗，進行螺旋式課程設計，並維持學生學習動機

✍短期記憶(工作記憶)：通常保持約20秒，通常記憶廣度約5-9個單位

人本主義取向的教育心理學

  • Maslow：
    • 人本主義心理學之父
    • 主張不用外鑠方式約束學生學習，學生具有內發的學習成長潛力，教師主要工作在布置良好學習環境
    • 提出需求層次論
    • 人具有兩股潛在力量(防衛、進取)不適當的教育會扼殺學生心靈上的生機
  • Combs：
    • 想改變學生的偏差行為不能單從行為上加以矯正，必須設法改變學生的知覺與信念
    • 教育目的主要在滿足學生情意方面需求
  • Rogers：
    • 提倡以學生為中心的教育理念
    • 教師應具備真誠一致、無條件積極關注與同理心的教學輔導技巧
  • Coopersmith
    • 培養學生自尊心
    • 「自我價值感」的三個條件
      1. 重要感
      2. 成就感
      3. 有力感：個人感覺到自己有處理事務與適應困境的能力(能獨立完成一件事情或指派的工作)
  • 卡芬頓(Covington)自我價值論
    • 個人追求成功的內在動力
    • 視成功為能力之展現非努力的結果
    • 成功難追求改以逃避失敗維持自我價值
    • 學生對能力與努力的歸因隨年級而轉移(認為努力而有好成績者，是因為其能力較低)

個別差異教育

認知類型

  • 指個人在面對問題情境時，會經由知覺、記憶、思維等內在心理歷程，表現出習慣性的外顯行為特徵
  • 榮格的分類
    • 感覺型
    • 直覺型
    • 思維型
    • 感情型
  • Witkin之分類
    • 場地獨立型：從事知覺判斷時，易把整體分析為若干成份，較少受周圍刺激物影響，能專注不易分心
    • 場地依賴型：將整體視為知覺範圍，無法專注其中某一部分細節，易受周圍刺激的影響，待人接物有依創頃向

其他

✍ 教育心理學研究方法

  • 橫段法(=橫切式)
    • 同一時間測量不同年齡之研究對象
    • 研究時間較縱貫式設計短
    • 樣本流失情形較縱貫式設計少
  • 縱貫法(=縱切法)
    • 對相同的個人或團體，在其不同的年齡或發展階段，連續觀察其發展情形
  • 創造能力的特徵(Torrance提出)
    • 變通性：舉一反三.觸類旁通
    • 獨特性
    • 流暢性：心智活動多流暢、少阻塞，能在短時間表達出較多的觀點
    • 精進性：較注重細節，注意事物的細膩部分，思考較為縝密
  • 擬情(=同理心)：是個人中心法的主要處理方式
  • 記憶的提取
    • 再認法recognition：要求學習者對學習過的資料重現眼前時，指認其項目是否在先前出現過；或者從一些選項中選出正確的訊息，如選擇、是非、填充、配合。
    • 重習法relearning：由艾賓豪斯Ebbinghaus所創，又叫節省法。是在測量個體重新學習其所學習過的事物之效率，通常二次學習所花時間較少，節省分數公式：[(初次練習次數─再練習次數)/初次練習次數]X100。
    • 回憶法recall：可分自由回憶、序列式兩種。要個體看一系列的項目，隔一段時間要求將先前所看之項目回憶出來，如填空、解釋名詞、問答題等，練習不夠、時間過長，會影響記憶量
      1. 自由回憶free recall method：對於材料的回憶之順序並沒有特別之規定。如回憶美國50州的名稱、身體的系統等。
      2. 序列式serial recall method：依照學習材料時所使用的順序或依據某種特定之順序來回憶。如依原子量排列的週期表。
