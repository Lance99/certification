教學設計

教學設計模式

  • 線性模式(ASSURE模式)

    • Analyze learners 分析學習者
    • State objecyive 陳述學習目標
    • Select media and materials 選擇教學媒體與教材
    • Utilize media and materials 使用教學媒體與教材
    • Require learner participation 要求學習者的參與
    • Evaluation 進行教學評量及修正

  • 建構觀的模式(R2D2)

    • R2D2(循環反省設計與發展模式)：Recursive Reflezive Design & Development Model
    • 三特點：
      • 界定焦點(未提教學目標)
      • 設計與發展焦點
      • 傳播焦點(沒有總結性評量，鼓勵學生自己設定個人目標進行不同學習活動)

  • ARCS模式(Keller所提出，引發學習動機的模式)

    • Attention 引起學習者的注意
    • Relevence 滿足學習者的個別需求
    • Confidence 建立學習者完成學習活動的動機與信心
    • Satisfaction 讓學習者從學習成果中獲得內在自我肯定

偏向技能領域的教學法

個別化取向教學法(偏向行為主義)

  • 編序教學法(循序自學法、計畫學習法)[Skinner提出]
    • 先將教材分析成許多細項，前一細目引導後一細目使學生循序依自身能力學習，可適應學生個別差異，培養學生自學態度和能力
    • 編序教學法非常適合用來做差異化教學或電腦輔助教學，學生可以依照自己的進度學習
  • 精熟教學法[Bloom所提倡]
    • 精熟教學法的5個目標
      • 將教材區分為多個小單元，每一或二週教完一個單元便舉行測試
      • 針對教學目標，使每次測驗的題目不要太難，使所有學生都能達到80~90%精熟地步
      • 每次測驗後都讓學生核成績，學生成就達不到精熟地步者，則給予額外時間補習(個人或團體)，補習之後再行測試，直到精熟再進行下一單元學習
      • 學生中學習能力較快者，在等待時間施予充實教學，以擴大或提升其知識的範圍或學習能力
      • 在學期結束舉行期末考試，試題包括全學期學過而且也學會的多個小單元
    • 精熟學習重要步驟如下
      • 設定明確可評量的教學目標：使全部學生或絕大多數(95%以上)學生精熟80~90%教材內容
      • 教師準備二份範圍和難度相同的測驗：一份叫無錯誤測驗作為形成性評量；另一份期末測驗作為總結性評量，以決定學期成績
      • 教師只要準備
        1. 導正性(補救)教學為未達精熟的學生而設
        2. 充實性活動為學習快速學生而設。
  • 個人化系統教學法(PSI)[Keller提出]
    • 行為主義，編序教學法+精熟學習+助理制(proctoring)
    • 特徵：
      • 成績好的學生擔任助理
      • 90％的熟練度
      • 學生全程自學(老師通常不上課)
      • 學生自我控速
      • 評量採標準參照
      • 及時增強
      • 教師講述功能為激發學習動機
      • 低挫折感
      • 適性教學
    • 過程(步驟)：
      1. 分多個小單元，並各自訂出明確目標
      2. 學生按自己步調學習，學畢後回到班上參加測試，測試後立即獲得結果，並從回饋中得到強化。
      3. 學生學習快速者，可充當輔助員角色，從旁輔助落後的學生
      4. 個單元測試通過後，其分數和代表該學生的學習成績
      5. 教學全程以自學輔導為主，教師很少講解課程，多於討論學習方法和鼓舞士氣
  • 自學輔導法

合作取向教學法

  • 協同教學法
  • 合作學習教學法
    • 組間同質，組內異質的分組原則
    • 學生小組成就區分法(STAD)
      • Student Teams-Achievement Divisions
      • STAD所進行的是個別測驗，而不是遊戲。每位學生要獨立完成自己的測驗，作為個人成績以及納入組別成績(個人進步的情形作為小組計分的依據)
    • 小組競賽法(TGT)
      • Teams-Games-Tourament
      • 競賽時將各組能力相當的人集中在同一競賽桌競賽，但不告訴學生各桌題目的難度水準。小組遊戲競賽法的小組學習時採異質性學習，以學生遊戲競賽代替小考
    • 拼圖法第一代 [由Elliot Aronson設計]
      • 全班授課→分組學習→進行小考
    • 拼圖法第二代 (參考STAD而加入計算個人成績並進行小組表揚)
      • 全班授課→分組學習→進行小考→小組表揚
    • 拼圖法第三代 (在小考之前加入了進行合作複習測驗)
      • 全班授課→分組學習→考前合作複習→進行小考→小組表揚
    • 拼圖法第四代 (在合作複習測驗之前加入了教師回饋)
      • 全班授課→分組學習→教師回饋→考前合作複習→進行小考→小組表揚

✍ 學習型組織應包括五項要素

  • 建立共同願景（Building Shared Vision）
  • 團隊學習（Team Learning）

  • 自我超越（Personal Mastery）

✍ Kilpatrick同時學習原則

  • 主學習：直接達到的教學目的
  • 副學習：與功課有關的思想或概念(如鴉片戰爭一課，同時學到地理和衛生之事)
  • 附學習(輔學習)：學習時所養成的理想態度(如實驗課學到科學精神)

✍ 課程行動研究

  • 是指針對課堂上的實際教學問題進行探討的一種研究，目的在於解決課程發展實務問題(課堂中教學遇到的困難)

✍ 雙向細目表

  • 主要在研究教學目標和教學內容

✍ Spencer：教育的目的在預備完美的生活

  • 五大目的：
    1. 衛生保健：與自我生存有直接關係的活動---身體的保健
    2. 職業技能：與自我生存有間接關係的活動---謀生的職業
    3. 家庭親職：關於繁殖種族的活動---做父母的準備
    4. 公民教育：關於維持社會關係和政治關係的活動---公民的道德活動
    5. 休閒教育：關於利用休閒時間和滿足趣味的活動---休閒和娛樂

✍ Doyle：【班級知識】是教師研究和班級實務的核心基礎

✍ 正義

  • 分配性正義：能力強報酬多，功績社會
  • 補償性正義：能力弱補償多
  • 需求的正義：重要是讓他們的需求得到同樣的滿足

✍ ORID (焦點討論法)

4個層次的溝通方式，讓人們有機會用對的順序，詢問對的問題，讓被討論的話題可以聚焦【尤其可用在團體討論時】

  • Objective：觀察外在客觀、事實【看到了什麼？發生了什麼事？】
  • Reflective：重視內在感受、反應【什麼是你覺得比較困難/容易/處理的？有什麼地方讓你很感動/驚訝/難過/開心？】
  • Interpretive：詮釋意義、價值、經驗【為什麼這些讓你很感動/驚訝/難過/開心？對你而言，重要的意義是什麼？學到了什麼？】
  • Decisional：找出決定、行動【有什麼我們可以改變的地方？接下來的行動/計劃會是什麼？】

✍ 試題鑒別度與難度最佳範圍

  • 難度最佳範圍：0.5~0.625最佳
  • 鑒別度最佳範圍：0.4以上最佳

✍ 選擇題相較於其他簡答題等考試形式更適用於各種不同層次學習結果的測量

✍ 提問原則與技巧

  • 轉引：運用一個具有多種答案的問題，加以探討的技巧
  • 轉問：在某位學生發問後，教師不直接回答該學生，而是轉問其他學生，以使更多學生參與討論，使學習重心在學生身上
