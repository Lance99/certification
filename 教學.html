教學目標

  • 具體目標(行為目標)：具體、可觀察、可測量的，通常是強調學習歷程完成後學生該表現的行為

  • 表意目標：不在意學生從教育活動後該展示的行為結果，而是在意學生所經歷的情景

  • 奇伯樂(Kibler)行為目標的敘寫

    • 行為的主體
    • 實際的行為
    • 行為的結果或內容：例如：一篇文章、小說
    • 行為的條件
    • 成功與否的標準：例如：答對了百分之八十五

      例如：學生在地球儀上，指出北美洲五大湖中三個湖的位置

      • 主體：學生
      • 行為：指出
      • 條件：在地球儀上
      • 標準：五大湖中的三大湖
      • 結果：湖的位置

教學設計模式

I. 線性模式(ASSURE模式)

  • Analyze learners 分析學習者
  • State objectives 陳述學習目標
  • Select media and materials 選擇教學媒體與教材
  • Utilize media and materials 使用教學媒體與教材
  • Require learner participation 要求學習者的參與
  • Evaluation 進行教學評量及修正

II. 建構觀的模式(R2D2)

  • R2D2(循環反省設計與發展模式)：Recursive Reflective Design & Development Model
  • 三特點：
    • 界定焦點(未提教學目標)
    • 設計與發展焦點
    • 傳播焦點(沒有總結性評量，鼓勵學生自己設定個人目標進行不同學習活動)

III. ARCS模式 [Keller所提出，引發學習動機的模式]

  • Attention 引起學習者的注意
  • Relevence 與學習者切身相關滿足個別需求
  • Confidence 建立學習者完成學習活動的動機與信心
  • Satisfaction 讓學習者從學習成果中獲得內在自我肯定

  • Raths的價值澄清法(CPA)
    • 價值形過程(重視過程而非結果)
      1. Choosing(選擇)：自由地和充分考慮地從各種可能選擇中抉擇

四、個別化取向教學法(偏向行為主義)

I. 編序教學法(循序自學法、計畫學習法) [Skinner提出]

  • 先將教材分析成許多細項，前一細目引導後一細目使學生循序依自身能力學習，可適應學生個別差異，培養學生自學態度和能力
  • 編序教學法非常適合用來做差異化教學或電腦輔助教學，學生可以依照自己的進度學習
  • 教材不易編製

II. 精熟教學法 [Bloom所提倡]

  • 精熟教學法的5個目標
    • 將教材區分為多個小單元，每一或二週教完一個單元便舉行測試
    • 針對教學目標，使每次測驗的題目不要太難，使所有學生都能達到80~90%精熟地步
    • 每次測驗後都讓學生核成績，學生成就達不到精熟地步者，則給予額外時間補習(個人或團體)，補習之後再行測試，直到精熟再進行下一單元學習
    • 學生中學習能力較快者，在等待時間施予充實教學，以擴大或提升其知識的範圍或學習能力
    • 在學期結束舉行期末考試，試題包括全學期學過而且也學會的多個小單元
  • 精熟學習重要步驟如下
    • 設定明確可評量的教學目標：使全部學生或絕大多數(95%以上)學生精熟80~90%教材內容
    • 教師準備二份範圍和難度相同的測驗：一份叫無錯誤測驗作為形成性評量；另一份期末測驗作為總結性評量，以決定學期成績
    • 教師只要準備
      1. 導正性(補救)教學為未達精熟的學生而設
      2. 充實性活動為學習快速學生而設
        Bloom：教學是先採團體教學，未精熟者才接受「補救教學」

III. 個人化系統教學法(PSI) [Keller提出]

  • 行為主義，編序教學法+精熟學習+助理制(proctoring)
  • 特徵：
    • 成績好的學生擔任助理
    • 90％的熟練度
    • 學生全程自學(老師通常不上課)
    • 學生自我控速
    • 評量採標準參照
    • 及時增強
    • 教師講述功能為激發學習動機
    • 低挫折感
    • 適性教學
  • 過程(步驟)：
    1. 分多個小單元，並各自訂出明確目標
    2. 學生按自己步調學習，學畢後回到班上參加測試，測試後立即獲得結果，並從回饋中得到強化。
    3. 學生學習快速者，可充當輔助員角色，從旁輔助落後的學生
    4. 個單元測試通過後，其分數和代表該學生的學習成績
    5. 教學全程以自學輔導為主，教師很少講解課程，多於討論學習方法和鼓舞士氣

五、合作取向教學法

I. 協同教學法

II. 合作學習教學法

組間同質，組內異質的分組原則

A. 學生小組成就區分法(STAD) Student Teams-Achievement Divisions

STAD所進行的是個別測驗，不是遊戲。每位學生要獨立完成自己的測驗，作為個人成績以及納入組別成績(個人進步的情形作為小組計分的依據)

B. 小組競賽法(TGT) Teams-Games-Tourament

競賽時將各組能力相當的人集中在同一競賽桌競賽，但不告訴學生各桌題目的難度水準。小組遊戲競賽法的小組學習時採異質性學習，以學生遊戲競賽代替小考

C. 拼圖法 (共4代)
1. 拼圖法第一代 [由Elliot Aronson設計]

全班授課 → 分組學習 → 進行小考

2. 拼圖法第二代 (參考STAD而加入計算個人成績並進行小組表揚)

全班授課 → 分組學習 → 進行小考 → 小組表揚

3. 拼圖法第三代 (在小考之前加入了進行合作複習測驗)

全班授課 → 分組學習 → 考前合作複習 → 進行小考 → 小組表揚

4. 拼圖法第四代 (在合作複習測驗之前加入了教師回饋)

全班授課 → 分組學習 → 教師回饋 → 考前合作複習 → 進行小考 → 小組表揚

D. 菲利普66討論法

六人一組、六分鐘的原則，小組形成後，立即在一分鐘內選出各小組的組長和助理，然後教師在一分鐘內說明討論主旨和問題的範圍，接著各小組必須在六分鐘內獲得討論的結果。教師在此種小組討論的任務包括決定討論主題、安排小組成員、宣佈開始討論、從旁觀察或給予回饋等

E. 焦點討論法 (ORID)

用在團體討論時，讓人們有機會用對的順序，詢問對的問題

  1. Objective：觀察外在客觀、事實
  2. Reflective：重視內在感受、反應

    什麼是你覺得比較困難的？有什麼地方讓你很感動？

  3. Interpretive：詮釋意義、價值、經驗

    為什麼這些讓你很感動？對你而言，重要的意義是什麼？學到了什麼？

  4. Decisional：決定、行動

    有什麼我們可以改變的地方？接下來的行動是什麼？

班級經營

一、班級經營理論

Doyle：【班級知識】是教師研究和班級實務的核心基礎

I. Ginott模式(和諧溝通)

鑑賞式稱讚，避免評價式，和諧溝通，教師身教(良好示範)
  • 欣賞式稱讚(=鑑賞式讚美)：針對「實際行為」稱讚，例如：你把教室窗戶擦得很亮
  • 評價式讚美：針對「人格」稱讚，例如：你很認真練琴、你很有耐心解決問題

II. Kounin模式(有效動力班級管理)

漣漪效應，教師中心，全面掌控，同時處理，終止不當行為

III. Canter模式(果斷訓練)

破唱片教學，決斷力教師，決斷力可後天培養，明確表達教師期望

IV. Jones模式(正向紀律)

善用肢體語言，有效幫助，以適當行為取代不當行為

V. Dreikurs模式(目標導向)

學生行為來自：獲得注意、尋求權力、報復傷害、表現無能
教師以一對一方式，協助學生瞭解其行為背後的目標，以避免抱持上述的偏差行為

VI. 其他模式

行為改變模式、果斷訓育模式、溝通分析模式、團體動力模式、理性結果模式、現實治療模式、教師效能訓練模式

二、班級經營技巧

I. Gordon的教師效能訓練

你訊息是溝通的絆腳石【講道、提醒、揶揄、評斷、分析、詢問、安慰、提供解決方法、讚美、同意、故作幽默...】

反對使用賞罰

A. 「I-message」策略(人本主義)
  • 我訊息包括三個部分
    1. 把困擾你的行為描述出來

      例如：我在深夜還聽到打電腦的聲音...

    2. 說出你對這件行為可能導致的結果有什麼感受

      例如：我在深夜還聽到打電腦的聲音，無法安心睡覺...

    3. 把後果或是對你所造成的影響說出來

      例如：我在深夜還聽到打電腦的聲音，無法安心睡覺，第二天會精神不濟...

  • 我訊息的做法
    1. 不會告訴學生如何去改變行為，由學生決定
    2. 不提到任何人的事情，集中在說話者身上
    3. 信任和相互尊重的口氣
B. 積極聆聽學生問題
  • 專注的沉默(身體姿勢、肢體語言)
  • 曖昧的回應：喔！糟了、你不說的話、我了解、這不是開玩笑的
  • 敲門磚：你要不要談一談、讓我們聽聽看你還有什麼話要說？
  • 積極的傾聽：最有效之一，聆聽當事人並且以當下反應來推測他內心的想法和感受
C. 採用雙贏策略解決問題(即：無人輸)
  • Gordon 衝突解決的六步驟：
    1. 定義問題：以積極傾聽來了解對方需求、以我訊息來表達自我需求，用需求定義問題，而不是用解決方法定義問題
    2. 可能的解決方法：腦力激盪種創意的方法
    3. 評估解決方法(能滿足雙方需求的)
    4. 選擇一個解決方案：避免表決、尋求共識
    5. 實施解決方法：同意做，用我訊息
    6. 實施後的評估：回去檢查，有效最好、無效重來(不要當成秘密，要公開)

其他

  • 正義
    • 分配性正義：能力強報酬多，功績社會
    • 補償性正義：能力弱補償多
    • 需求的正義：重要是讓他們的需求得到同樣的滿足
  • 提問原則與技巧
    • 轉引：運用一個具有多種答案的問題，加以探討的技巧
    • 轉問：在某位學生發問後，教師不直接回答該學生，而是轉問其他學生，以使更多學生參與討論，使學習重心在學生身上
  • Glasser的選擇理論
    • 學習行為是一種選擇，在於滿足內在的心理需求，不是外塑的。如果學習行為能滿足學生的各項需求，則學生必然會自動自發的去學習
  • Jennings 的四種協同教學模式
    1. 兩人分工 一位教師介紹教學內容，而另一位執行各種活動(最常見的模式)
    2. 分站式 教室內規劃數個教學中心，學生輪流至各站參與站長指導的學習。若無站長指導，可發展成為學生探索的活動

      此方法具有四項優點：

      1. 在學習中心之間移動，能改變學生活動，避免疲累
      2. 減少每組人數，有助成員互動及扮演主動角色
      3. 教室內教材不足，可採此種方式教學
      4. 教師計畫時間不足，利用此方法可減少規劃時間
    3. 目錄式 教師協同設計多種活動內容，讓學生掌握學習的重要內涵及自主學習的機會，可參考Gardner的多元智能理論規劃內容架構。本方式是用於無站長的學習中心、學生可以自主選擇的學習活動
    4. 並列式　 把學生分成人數相同的兩組，各自教授相同的內容；也可以各自教相同內容，但在複雜度或方式上有所差異。其優點包括：教師可以主動規劃課程；學生分組後人數少，易於關注學生個體；而且教師容易規劃
  • Kolb經驗學習的4階段過程
    1. 始於個人的具體經驗(例如：參與淨灘)
    2. 為個人進行觀察和省思(例如：省思與發表心得)
    3. 獲得抽象概念並加以概括類化(例如：獲得環保意識+為活動做總結)
    4. 最後在新情境中檢驗概念的正確性。
  • 教育機會均等
    • 歷程的均等：公平對待不同身分背景、階層的學生
    • 資源的均等：給予弱勢學生較多資源(垂直公平)
    • 輸入的均等：進相同類型學校教育
    • 結果的均等：提供低成就學生補救教學
  • Torrance 創造思考測驗的評量指標
    • 流暢力：思考靈活順暢，能提出多種可行想法
    • 變通力：觸類旁通的能力
    • 獨創性：獨特的見解，推陳出新的能力
    • 精進力：精益求精的能力
  • Guilford 的創造力五力
    • 流暢力
    • 變通力
    • 精進力
    • 敏覺力
    • 獨創力
  • 心向

    定義：心向指心之所向，對於一件事的做法，常採用同一方法，久之成為習慣

    1. 使用過去解決類似問題獲得成功的方式解決當前的問題
    2. 不管其他方法是否有效，人會用特定的方式來解決問題(針對事情解決的功能固著）
    3. 無論客觀條件為何，總是以其主觀經驗或習慣去處理問題
    4. 是非題的心向程度比選擇題強
  • 精緻化：讓學習者將所要學習的新訊息與已習得的舊訊息產生相互聯結【例如：引申、舉例、類比、重述、做摘要、字鉤法、聯想法等】
  • 組織：將訊息加以歸類，使相關的新舊訊息間產生密切的聯結【例如：做大綱、畫概念關係圖】
  • 逃避精熟目標：避免犯錯，逃避學習更深的知識
  • 逃避表現目標：維持公眾形象，避免無能之指責
  • 教育機會均等
    • 過程均等：相同的教育內容
    • 結果均等：相同的教育成就
    • 輸入均等：給弱勢多一點資源
